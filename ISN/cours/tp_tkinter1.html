Créer une interface graphique (des fenêtres, boutons, etc. avec lesquels on peut intéragir.) est fastidieux et relativement complexe. Nous allons nous «contenter» d’utiliser celle proposée par la majorité des distributions Python avec le module tkinter. (Tool Kit interface)

Exercice 1: Première fenêtre

Commençons par créer notre première fenêtre, en testant le code suivant:

from tkinter import *

fenetre = Tk()

fenetre.mainloop()

Le module Tkinter permet de réaliser de la programmation événementielle. Ce type de programme est basé sur une «boucle infinie» qui permet d’attendre l’apparition des évènements. (clic sur la souris, déplacement de la souris, appui sur une touche du clavier…​)
C’est l’instruction mainloop() qui permet de lancer cette «boucle infinie», et qui viendra donc toujours à la fin du programme après avoir définie l’aspect graphique et la réaction aux évènements.

Exercice 2: Premiers widgets

Nous pouvons ajouter dans notre fenêtre différents éléments graphiques, des widgets (windows gadgets), tels que des boutons ou des listes déroulantes…​

Par exemples:

  • Les widgets Button permettent d’afficher un bouton avec un texte.

  • Les widgets Label permettent d’afficher du texte.

  • Les widgets Entry permettent à l’utilisateur d’entrer du texte.

La syntaxe pour créer un widget est:

w = Widget(conteneur, opt1=val1, opt2=val2, ...)

Tout widget doit être positionné dans son conteneur. La méthode pack() permet de placer les widgets dans leur conteneur et d’adapter la taille.

La syntaxe est:

w.pack(opt1=v1, opt2=v2, ...)

Tester l’exemple suivant:

from tkinter import *

fenetre = Tk()
fenetre.title("Qui es-tu ?")

mon_bouton = Button(fenetre, text="Ok", width=30, command = fenetre.destroy)
mon_texte = Label(fenetre, text="Entrez votre nom ici:", fg="red", bg="blue")
ma_saisie = Entry(fenetre)

mon_texte.pack()
ma_saisie.pack()

fenetre.mainloop()

  1. Modifier le programme pour que le bouton apparaisse, sous l’espace de saisie.

  2. Tester le bouton.

  3. A quoi correspond fg ? bg ? width ?

Exercice 3: Le widget Canvas (canevas): introduction

Un canevas définit une zone dans laquelle nous pourrons «dessiner» et afficher des images.

Qu’entend-on par «dessiner» ? En fait, des instructions vont nous permettre de représenter des figures tels que des segments, des rectangles, des ellipses.

Par exemple, on peut tracer en rouge le segment reliant les points de coordonnées \((x_1; y_1)\) et \((x_2; y_2)\), sur le canevas Can, en utilisant la commande suivante:

Can.create_line(x1,y1,x2,y2,fill="red")

Tester le code suivant:

from tkinter import *
HAUTEUR = 400
LARGEUR = 400
Fenetre = Tk()

Can = Canvas(Fenetre,bg="#000000FBD", height=HAUTEUR, width=LARGEUR)
Can.pack()

Can.create_line(0,0,HAUTEUR,LARGEUR,fill="red")
Can.create_line(HAUTEUR//2,LARGEUR//2,0,LARGEUR//2,fill="green")
Fenetre.mainloop()

On remarque que le point en haut à gauche à pour coordonnée \((0; 0)\) et celui en bas à droite \((400; 400)\). (dans un canevas de dimension 400x400)

Ecrire un programme qui affiche dans un canevas carré de dimension 400x400, les diagonales du carré formé par le canevas.

Exercice 5: Plus de tracés.

Dessiner un rectangle:

Can.create_rectangle(x1,y1,x2,y2,fill="black")

Paramètres:

  • outline: couleur du bord.

  • fill: couleur de l’intérieur.

Le point \((x_1; y_1)\) représente le coin haut gauche du rectangle et \((x_2; y_2)\) le coin en bas à droite.

Dessiner une ellipse/un cercle:

Can.create_oval(x1,y1,x2,y2, outline="blue")

Paramètres:

  • outline: couleur du bord.

  • fill: couleur de l’intérieur.

Les points \((x_1; y_1)\) et \((x_2; y_2)\) correspondent aux coordonnées du rectangle circonscrit (qui contient l’ellipse)

Pour tracer un cercle de rayon \(r\) et de centre \((x; y)\), on peut donc utiliser:

Can.create_oval(x-r,y-r,x+r,y+r, outline="black")

Dessiner du texte:

Can.create_text(x, y, text="mon texte ici", font="Arial 12")

Paramètres:

  • fill : couleur de l’écriture

  • x,y : centre du texte à écrire

Dessiner un polygone:

Can.create_polygon(x1,y1,x2,y2,x3,y3)

Paramètres:

  • fill: couleur de l’intérieur.

  • outline: couleur du bord.

  • smooth=True pour arrondir les lignes (=False, pour ne pas arrondir..)

On donne les coordonnées d’autant de points que voulu et ces points sont reliés dans l’ordre donné, et le dernier est relié avec le premier.

Ecrire un programme qui affiche dans un canevas de dimension 800x600:

  1. Un cercle rouge de rayon 50 pixel et centré au milieu du canevas.

  2. Un rectangle de longueur 100 pixel, et de hauteur 50 pixel, et dont le coin haut gauche se trouve au pixel de coordonnées \((150; 500)\).

  3. Le message: "La géométrie, c’est super!". Ce message doit être situé à 50 pixels du haut du canevas et centré sur la largeur.

Exercice 6: Insérer une image/gestion de la souris

Un des principaux intérêt des canevas, c’est qu’ils permettent d’insérer une image.

Attention cependant tout les formats ne sont pas acceptés et le principal format accepté est le .gif

La syntaxe:

ma_photo = PhotoImage(file="nomfichier")
Can.create_image(x,y, image=ma_photo)

\((x; y)\) correspond à la position du centre de l’image.

Télécharger l’image cibles.gif dans le répertoire de votre programme Python.

Et afficher cette image dans un canevas de dimension 800x800.

On peut lier un clic sur la souris à l’appel d’une fonction à l’aide de la méthode bind.

Exemple:

from tkinter import *
import time
Fenetre = Tk()
HAUTEUR = 800
LARGEUR = 800

Can = Canvas(Fenetre,bg="#FFF", height=HAUTEUR, width=LARGEUR)
Can.pack()

def tir(event):
    r = 5
    X = event.x
    Y = event.y
    Can.create_oval(X-r,Y-r,X+r,Y+r,fill="black")

Fenetre.bind("<Button-1>",tir)
Fenetre.mainloop()

  1. Modifier l’exemple précédent pour qu’il permet de tirer sur la cible.

  2. Modifier le même exemple, pour qu’un clic droit affiche un cercle rouge, trois fois plus large que le tir clic gauche.

Les noms des différents boutons de la souris:

  • <Button-1> : click sur le bouton gauche de la souris

  • <Button-2> : click sur la molette de la souris

  • <Button-3> : click sur le bouton droit de la souris

  • <Double-1> : double-click gauche

Exercice 7: Récupérer/modifier une option d’un widget

Récupérer la valeur d’une option opt d’un widget w

w["opt"]

Modifier une option

w["opt"] = valeur

Exemple, à tester:

from tkinter import *
fenetre = Tk()
mon_texte = Label(fenetre, text="Bonjour !", fg="red", bg="blue")
mon_texte.pack()

mon_texte["text"] = "Au revoir !"
fenetre.mainloop()

Le widget Button possède une option command, qui précise la fonction à lancer lors d’un clic sur ce bouton.

from tkinter import *
fenetre = Tk()

def interrupteur():
  if mon_bouton["text"] == "On":
    mon_bouton["text"] = "Off"
  else:
    mon_bouton["text"] = "On"

mon_bouton = Button(fenetre, text="On", width="20", height="10", command=interrupteur)
mon_bouton.pack()

fenetre.mainloop()

  1. Tester cet exemple.

  2. Modifier ce programme, pour que le texte soit en blanc sur fond noir, lorsque le bouton affiche on, et noir sur fond blanc, lorsque le bouton affiche off.
